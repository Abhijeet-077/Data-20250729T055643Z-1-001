How to Use the Interactive Trade Execution Analysis

×

Parameters Explanation:

  • Sample Size: Number of data rows to analyze (100-5000). Higher values provide more accurate models but take longer to process.
  • Order Size Points: Number of data points for slippage analysis (10-50). More points give smoother curves.
  • Max Book Depth: Maximum percentage of order book depth to consider (10-100%). Higher values simulate larger orders.
  • Total Shares: Total number of shares to allocate across trading intervals.
  • Trading Intervals: Number of time periods to split the trade (5-20).

Features:

  • Interactive Charts: Zoom and pan on charts using mouse wheel and drag.
  • Real-time Analysis: Parameters are applied immediately when you click "Run Analysis".
  • Export: Download results in JSON and CSV formats.
  • Comparison: Select "Compare All Tickers" to see side-by-side analysis.

Risk Metrics:

  • Total Slippage Cost: Expected total cost of market impact.
  • Max/Min Allocation: Largest and smallest trade sizes in the optimal allocation.
  • Allocation Std Dev: Measure of how varied the trade sizes are.
